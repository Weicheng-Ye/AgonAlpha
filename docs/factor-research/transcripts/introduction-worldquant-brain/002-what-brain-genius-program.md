## Description

The BRAIN Genius program is a leveled system for WorldQuant BRAIN consultants that provides increasing access to data, tools, and opportunities based on performance and activity levels. There are four levels: Gold (default level for new BRAIN consultants), Expert, Master, and Grand Master. These levels are an extension to the initial levels of users that you are already familiar with: Bronze, Silver and Gold.
